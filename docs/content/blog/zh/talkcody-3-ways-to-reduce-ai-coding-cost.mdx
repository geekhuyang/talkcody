## 第1招：复用你已购买的订阅

这是最直接、最省钱的方案。如果你已经订阅了 Claude Pro/Max、ChatGPT Plus/Pro 或 GitHub Copilot，通过 TalkCody 直接复用这些订阅，**零额外成本**。

### 1.1 Claude Pro/Max 订阅复用

如果你已经订阅了 Claude Pro（$20/月）或 Claude Max（$100-200/月），通过 TalkCody 的官方 OAuth 认证，可以直接使用订阅内的额度。

**使用方式：**
1. 打开 TalkCody 设置 → API Keys → Anthropic (Claude)
2. 点击"使用 Claude 登录"
3. 在浏览器中完成授权，粘贴授权码
4. 开始使用 Claude Sonnet、Claude Opus 等模型

**优势：**
- 无需额外购买 API，使用现有订阅
- 实时使用量仪表板，监控 5 小时和 7 天使用情况
- 自动令牌刷新，无需重复登录

### 1.2 ChatGPT Plus/Pro 订阅复用

同样，如果你已经订阅了 ChatGPT Plus（$20/月）或 ChatGPT Pro（$200/月），也可以通过 TalkCody 直接复用。

**使用方式：**
1. 打开 TalkCody 设置 → API Keys → OpenAI
2. 点击"使用 OpenAI 登录"
3. 在浏览器中完成授权
4. 开始使用 GPT-5.2、GPT-5.2-Codex 等模型

**优势：**
- 复用现有订阅，无需额外 API 费用
- 使用最新的 GPT-5.2 系列模型
- 自动令牌刷新，体验流畅

### 1.3 GitHub Copilot 订阅复用

GitHub Copilot 也是一个不错的选择，尤其是如果你已经在使用 Copilot。

**使用方式：**
1. 打开 TalkCody 设置 → API Keys → GitHub Copilot

## 第2招：使用 Coding Plan 高性价比方案

Coding Plan 是 MiniMax、智谱 AI 和 Moonshot 提供的**编程专属订阅计划**，专为高频编码场景设计，相比直接按 API 计费，可以节省大量成本。

### 2.1 什么是 Coding Plan？

Coding Plan 提供商及其支持模型：

| 模型 | 提供商 | 特点 |
|------|--------|------|
| MiniMax M2.1 | MiniMax | 高性能国产大模型 |
| GLM 4.7 | 智谱 AI | 专为编码优化 |
| Kimi (K2) | Moonshot | 月之暗面推理模型 |

这些模型都有专门的 Coding Plan API 端点，针对编程任务进行了优化。

### 2.2 Coding Plan 优势

相比直接购买 API，Coding Plan 的优势明显：

**1. 性价比极高**
- 订阅模式，固定成本，不限调用次数
- 相比按量计费，高频使用场景下可节省 50-80% 费用

**2. 编程任务优化**
- 专门针对编程场景的 API 端点
- 响应更准确，代码生成质量更高

**3. 内置 MCP 服务器**
TalkCody 内置了 MiniMax M2.1 和 GLM 4.7 的 MCP 服务器，提供：
- **网络搜索**：搜索最新技术文档、API 参考、编程问题
- **图像识别**：分析错误截图、理解 UI 设计稿、解读流程图

配置 Coding Plan 后，API Key 会自动同步到 MCP 服务器，无需额外配置。

### 2.3 配置简单

只需 3 步即可开始使用：

1. **获取 API Key**
   - MiniMax: 访问 [开放平台](https://platform.minimaxi.com/docs/coding-plan/intro)
   - 智谱 AI: 访问 [开放平台](https://docs.bigmodel.cn/cn/coding-plan/overview)
   - Moonshot: 访问 [开放平台](https://www.kimi.com/coding/docs/)

2. **配置 API Key**
   - 打开 TalkCody 设置 → API Keys
   - 填入对应提供商的 API Key

3. **开启 Coding Plan**
   - 勾选"Use Coding Plan"开关
   - 保存设置即可使用
